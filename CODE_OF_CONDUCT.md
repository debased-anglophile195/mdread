# Code of Conduct

mdread is a small, friendly open-source project. To keep it welcoming, everyone
taking part — issues, pull requests, discussions — is expected to be respectful
and considerate.

## The short version

- Be kind and assume good faith.
- Critique ideas and code, not people.
- Respect differing viewpoints and experiences.
- No harassment, personal attacks, or unwelcome behavior of any kind.

## Standard we follow

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1. Please read it for the full standard, including the kinds of behavior
that are encouraged and those that are unacceptable, and the responsibilities of
maintainers.

## Reporting

If someone is behaving inappropriately, or you have any concern, contact the
maintainer privately at **cep.jewel@gmail.com**. Reports will be reviewed and
handled discreetly. Maintainers may remove or reject contributions, comments, and
contributors that don't follow this Code of Conduct.

This applies in all project spaces and when representing the project in public.
